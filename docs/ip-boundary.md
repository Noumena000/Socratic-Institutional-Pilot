# Intellectual-Property Boundary

This public institutional pilot demonstrates a workflow and user experience. It does not distribute the private Socratic reasoning engine.

Excluded proprietary components include pedagogical policy logic, learner-state transition rules, prompt orchestration, evidence-update algorithms, private evaluation datasets, production schemas, and institution-specific integrations.

A future approved deployment should connect through a narrow, documented interface and a separately governed private adapter. Any institutional collaboration should define background intellectual property, institution-owned materials, permitted use, confidentiality, and ownership of general-purpose improvements in writing.
