# Relationship to the Socratic Project Family

## Socratic Assessment Platform

The public assessment showcase communicates the general educational thesis and public-facing product vision.

## Socratic Reasoning Platform

The private reasoning platform develops the provider-independent pedagogical engine, learner-state model, evidence logic, and offline/online orchestration.

## Socratic Institutional Pilot

This repository demonstrates how the educational workflow could appear in an institution-oriented setting using fictional data and a replaceable approved-AI boundary. It should consume a narrow interface to the reasoning core rather than duplicate or disclose that core.
