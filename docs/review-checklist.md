# Foundation Review Checklist

- [ ] Public language does not imply institutional approval or endorsement.
- [ ] All names, dialogue, courses, and records are fictional.
- [ ] No production credentials, endpoints, or learner information are present.
- [ ] The private reasoning engine is not copied into the repository.
- [ ] Faculty authority and uncertainty are stated clearly.
- [ ] The demonstration works with keyboard navigation and reduced motion.
- [ ] The scripted workflow accurately represents the proposed reasoning sequence.
- [ ] The next functional milestone is agreed before connecting a real AI provider.
