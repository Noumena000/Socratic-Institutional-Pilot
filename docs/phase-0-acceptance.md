# Phase 0 Acceptance Criteria

Phase 0 is complete when the repository clearly communicates all of the following:

- the project is an independent fictional institutional pilot;
- a language model is a replaceable provider rather than the assessment authority;
- the differentiated product is the educational intelligence layer;
- faculty define learning objectives and retain final judgment;
- learner claims are intended to remain linked to transcript evidence;
- the public repository excludes the proprietary reasoning core and production integration details;
- no institution, learner data, or approved AI service is represented as connected;
- future provider access, authentication, storage, and LMS integration require separate approval and governance.

The current foundation satisfies these communication and architecture criteria. Functional validation belongs to Phase 1 and later phases.
