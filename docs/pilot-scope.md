# Pilot Scope

## Purpose

This project demonstrates how an institution could use structured Socratic dialogue to gather reviewable evidence of learner reasoning while preserving faculty authority over academic decisions.

## Demonstration boundary

The current repository uses fictional courses, fictional learners, and simulated provider responses. It is not an approved deployment and is not affiliated with or endorsed by any specific institution.

## Initial vertical slice

The first pilot scenario focuses on one bounded conceptual distinction:

- validity versus soundness;
- one initial learner response;
- one targeted Socratic follow-up;
- a visible evidence update;
- an instructor-facing decision snapshot.

## Included

- learner dialogue demonstration;
- understanding evidence linked to learner turns;
- faculty-review workflow;
- mock institution-approved AI provider;
- governance and architecture documentation.

## Excluded

- real learner data;
- authentication;
- production storage;
- LMS integration;
- grade passback;
- misconduct determinations;
- proprietary reasoning-engine implementation;
- production AI credentials or endpoints.

## Success criteria

A reviewer should be able to understand the proposed learner journey, see how evidence accumulates across turns, and distinguish the institutional demonstration from the private reasoning core.
