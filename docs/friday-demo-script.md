# Friday Demonstration Flow

## Opening

This is an independent institutional pilot demonstrating how structured dialogue could make learner reasoning more visible while faculty retain final academic judgment.

## Demonstration sequence

1. Introduce the fictional learner and bounded validity-versus-soundness assessment.
2. Start the dialogue and show the learner's initial conceptual distinction.
3. Point out that the platform identifies a specific evidence need rather than merely generating another generic question.
4. Show the targeted request for a constructed example.
5. Watch the evidence snapshot update from definition to application.
6. Emphasize that revision remains unobserved rather than being treated as demonstrated.
7. Explain that the current provider is simulated and could later be replaced by an institution-approved AI service through a secure adapter.
8. Close with the faculty-review requirement and proposed controlled-pilot pathway.
