# Current Demonstration Limitations

The first build is intentionally narrow.

- The dialogue is scripted rather than generated from live learner input.
- The provider adapter is a mock contract and is not wired into the browser flow yet.
- Evidence is illustrative and does not come from the private reasoning engine.
- There is no authentication, database, audit log, export, LMS connection, or grade workflow.
- The build should not be used with real learner information.

These limitations preserve a clear boundary while the educational workflow, institutional requirements, and approved AI integration pathway are reviewed.
