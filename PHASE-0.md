# Phase 0 — Foundation

Phase 0 establishes the institutional pilot's strategic and architectural foundation.

## Completed

- institutional problem statement;
- explicit distinction between language intelligence and educational intelligence;
- provider-independent system architecture;
- public/private intellectual-property boundary;
- governance, fictional-data, and faculty-review commitments;
- one bounded fictional assessment scenario;
- roadmap and readiness criteria for the functional demonstration.

## Phase 0 acceptance statement

The repository now explains why access to a capable AI service is not, by itself, an institutional assessment solution. It presents the Socratic Platform as the educational intelligence layer that supplies pedagogical policy, learner-state tracking, evidence provenance, provider independence, and faculty governance.

## Next phase

Phase 1 will turn the current scripted vertical slice into a fuller functional demonstration with visible learner-state transitions, a mock provider contract, an understanding map, and an expanded faculty evidence portfolio.
